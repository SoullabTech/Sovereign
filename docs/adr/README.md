# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the MAIA Sovereign project.

## What is an ADR?

An ADR is a document that captures an important architectural decision made along with its context and consequences.

## Creating a New ADR

1. Copy the template: `cp docs/adr/template.md docs/adr/NNN-title.md`
2. Fill in the sections
3. Submit for review
4. Update this index when accepted

## Status Lifecycle

- **Proposed** - Under discussion
- **Accepted** - Approved and in effect
- **Deprecated** - No longer applies (superseded or abandoned)
- **Superseded** - Replaced by another ADR (link to replacement)
