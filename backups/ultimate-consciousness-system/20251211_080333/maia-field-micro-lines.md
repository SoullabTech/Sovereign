- *"Your individual season doesn't match what I'm sensing in the broader community, which sometimes happens."*

### **When Individual Needs Individual Focus**

- *"This feels like a moment for individual attention rather than community context."*

## **Crisis Mode Micro-Lines**

### **When Field Context Might Help in Crisis**
- *"What you're experiencing isn't happening in isolation - there are others in the community navigating similar challenges."*
- *"The field has been heavy this week, which might be contributing to how intense things feel right now."*

### **When to Skip Field Context in Crisis**
- **Skip entirely** - focus on individual safety and support only

## **Implementation Guidelines**

### **When to Include Field Micro-Lines:**

‚úÖ **Use field micro-lines when:**
- Individual pattern clearly matches field pattern
- User has decent capacity for broader perspective
- It would reduce isolation or increase belonging
- It provides helpful context for timing/decisions

‚ùå **Skip field micro-lines when:**
- Individual is in crisis or very low capacity
- No clear field pattern match
- User explicitly wants individual-only focus
- Field context might increase overwhelm

### **Tone Guidelines:**

1. **Gentle and optional** - "You know..." "It seems like..." "There's something..."
2. **Belonging-focused** - "You're not alone..." "Many people here..."
3. **Normalizing** - "This makes sense because..." "No wonder you're feeling..."
4. **Brief** - 1-2 sentences maximum
5. **Natural** - Should feel like organic conversation, not inserted data

### **Placement in Conversation:**

**Early in response:**
- Sets context for individual guidance
- Reduces isolation before diving deeper

**Example flow:**
1. Acknowledge individual experience
2. Field micro-line (if appropriate)
3. Individual guidance and support

**Never:**
- Lead with field patterns over individual acknowledgment
- Use field patterns to override individual experience
- Reference field when user needs pure individual attention
